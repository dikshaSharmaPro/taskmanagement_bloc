# Task Management App

A premium and intuitive Task Management App built using **Flutter**, powered by **BLoC** for state management and directly integrated with **Supabase** for real-time backend operations.

---

## Features

- Add, edit, delete tasks with real-time sync
- Mark tasks as completed
- Select custom date & time for each task
- Sorting options (Newest, Oldest, Completed, Pending)
- Real-time data synchronization with Supabase
- Clean, dark-themed UI with glassmorphic elements
- Animated transitions and micro-interactions
- Pull to refresh

## Tech Stack

- **Flutter**
- **BLoC** (State Management)
- **Supabase** (Real-time DB, Auth)
- **Dart**
- **Intl** for date formatting
- **FL Chart** for task statistics
- Shared pref for local storage
